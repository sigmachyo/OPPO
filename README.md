### Практическая работа 1-2: Основы ООП. Классы и объекты

**Дата выполнения:** 2023  
**Файлы:** `pr1_2/Source.cpp`, `pr1_2/file.txt`

#### Цель
Изучение базовых принципов создания и использования классов в C++.

#### Задачи
- Объявление классов с полями и методами
- Конструкторы и деструкторы
- Модификаторы доступа (public, private, protected)
- Геттеры и сеттеры (инкапсуляция)
- Перегрузка методов и операторов

#### Ключевые концепции

**Инкапсуляция:**
```cpp
class Person {
private:
    string name;
    int age;
    
public:
    // Конструктор
    Person(string n, int a) : name(n), age(a) {}
    
    // Геттеры
    string getName() const { return name; }
    int getAge() const { return age; }
    
    // Сеттеры с валидацией
    void setAge(int a) {
        if (a > 0 && a < 150) age = a;
    }
};
```

**Перегрузка операторов:**
```cpp
class Complex {
private:
    double real, imag;
    
public:
    Complex operator+(const Complex& other) {
        return Complex(real + other.real, imag + other.imag);
    }
    
    bool operator==(const Complex& other) {
        return real == other.real && imag == other.imag;
    }
};
```

#### Результат
- Понимание принципа инкапсуляции
- Умение создавать классы с конструкторами
- Навыки работы с указателем `this`

### Практическая работа 3: Наследование и полиморфизм

**Дата выполнения:** 2024  
**Файлы:** `pr3/test.cpp`, `pr3/test_code.cpp`

#### Цель
Освоение механизмов наследования и полиморфизма в C++.

#### Задачи
- Создание базовых и производных классов
- Виртуальные функции и полиморфизм
- Абстрактные классы и чистые виртуальные методы
- Множественное наследование (базово)
- Приведение типов (dynamic_cast, static_cast)

#### Ключевые концепции

**Наследование:**
```cpp
class Animal {
protected:
    string name;
    
public:
    virtual void speak() {
        cout << "Some sound" << endl;
    }
    
    virtual ~Animal() {}  // Виртуальный деструктор
};

class Dog : public Animal {
public:
    void speak() override {
        cout << "Woof!" << endl;
    }
};

class Cat : public Animal {
public:
    void speak() override {
        cout << "Meow!" << endl;
    }
};
```

**Полиморфизм:**
```cpp
vector<Animal*> animals;
animals.push_back(new Dog());
animals.push_back(new Cat());

for (Animal* animal : animals) {
    animal->speak();  // Вызов переопределённого метода
}
```

**Абстрактный класс:**
```cpp
class Shape {
public:
    virtual double area() const = 0;  // Чисто виртуальный метод
    virtual ~Shape() {}
};

class Rectangle : public Shape {
private:
    double width, height;
    
public:
    double area() const override {
        return width * height;
    }
};
```

#### Результат
- Понимание иерархий классов
- Опыт работы с виртуальными функциями
- Умение использовать полиморфизм для гибкого кода

## 🎯 Приобретённые навыки

### ООП концепции
- **Инкапсуляция:** сокрытие внутренней реализации, геттеры/сеттеры
- **Наследование:** базовые и производные классы, protected доступ
- **Полиморфизм:** виртуальные функции, переопределение (override)
- **Абстракция:** абстрактные классы, интерфейсы

### C++ особенности
- **Конструкторы/деструкторы:** инициализация, очистка ресурсов
- **Перегрузка:** операторов и методов
- **Указатели и ссылки:** работа с памятью, передача параметров
- **STL контейнеры:** vector, string, map

### Работа с файлами
```cpp
#include <fstream>

// Чтение из файла
ifstream inFile("data.txt");
string line;
while (getline(inFile, line)) {
    cout << line << endl;
}
inFile.close();

// Запись в файл
ofstream outFile("output.txt");
outFile << "Hello, File!" << endl;
outFile.close();
```

### Обработка исключений
```cpp
try {
    // Код, который может выбросить исключение
    if (error) {
        throw runtime_error("Something went wrong");
    }
} catch (const runtime_error& e) {
    cerr << "Error: " << e.what() << endl;
} catch (...) {
    cerr << "Unknown error" << endl;
}
```

## 📊 Сравнение: Процедурное vs ООП

| Характеристика | Процедурное | ООП |
|---|---|---|
| **Основная единица** | Функция | Класс/Объект |
| **Подход** | Сверху-вниз | Снизу-вверх |
| **Данные** | Отдельно от функций | Инкапсулированы в объекте |
| **Расширяемость** | Сложная | Лёгкая (наследование) |
| **Применение** | Простые программы | Крупные проекты |
